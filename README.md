# Linux Portfolio

A modern, interactive portfolio website built with React, TypeScript, and Vite. This project showcases a Linux-themed portfolio with a unique terminal-like interface.

## 🚀 Features

- Modern and responsive design
- Terminal-like interface
- Smooth animations using Framer Motion
- Interactive scrolling navigation
- Font Awesome icons integration
- Bootstrap components
- TypeScript for type safety

## 🛠️ Tech Stack

- React 19
- TypeScript
- Vite
- Framer Motion
- Bootstrap
- Font Awesome
- TailwindCSS
- ESLint

## 📦 Installation

1. Clone the repository:
```bash
git clone https://github.com/atm98/Project-Portfolio.git
cd Project-Portfolio
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

4. Build for production:
```bash
npm run build
```

## 🎯 Usage

- `npm run dev` - Start the development server
- `npm run build` - Build the project for production
- `npm run preview` - Preview the production build locally
- `npm run lint` - Run ESLint to check code quality

## 🚀 Deployment

This project is automatically deployed to GitHub Pages using GitHub Actions. The deployment process:

1. Triggers on push to the `development` branch
2. Builds the project using `npm run build`
3. Deploys the built files to [atm98.github.io](https://atm98.github.io)

To deploy manually:
1. Push your changes to the `development` branch
2. Go to the Actions tab in your repository
3. Select the "Deploy to GitHub Pages" workflow
4. Click "Run workflow"
